Project 1:

Higher Lower

The game Higher Lower is a guessing game where the player plays against the computer. Both the player and the computer have 2 dice each and they need to guess if the other rolls higher or lower. You win the game when you reach 10 points or when the computer gets -10 points.


I made this project with: Html, Css and JavaScript

Project 2:

Wack The Mole

Wack the mole is a solo player game where you have to hit moles until you reach a certain amount of points in order to win. If you lose your lives, you lose the game. In this game, the points needed to win are 10,000 points and the player has 5 lives.


I made this project with: Html, Css and JavaScript

Project 3:

Review Your Experience

This project is an interactive website made for people who want to invest in crypto currencies. The website has the following pages: home page, an about us page, a contact page and a guides page.
